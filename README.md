## ğŸ® Tips Bermain Game

1. **Hiragana/Katakana Quiz**:
   - Ketik jawaban dalam romaji (a, ka, shi, chu, dll.)
   - Tidak perlu menekan Enter, jawaban benar otomatis lanjut

   - B+ = 85-89%
   - B = 80-84%
   - C+ = 75-79%
   - C = 70-74%
   - D = 60-69%
   - F = 0-59%
